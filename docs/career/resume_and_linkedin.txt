## ROLE: ML ENGINEER / AI ENGINEER
## (Shopify, Cohere, Vector Institute, RBC, TD, Wealthsimple, BenchSci)
## ─────────────────────────────────────────────────────────────────────

• Engineered a production NLP microservice classifying Twitter/X sentiment across 4 classes
  (Positive/Negative/Neutral/Irrelevant), fine-tuning DistilBERT to 92.1% weighted F1 on
  74,682 labelled tweets — a 9-point improvement over the Bi-LSTM baseline (83.2%)

• Designed a dual-model inference system with automatic fallback: DistilBERT (~48ms P50)
  as primary and Bi-LSTM (~12ms P50) as CPU-only fallback, achieving 99.9% availability
  under simulated load with zero code changes at the API layer

• Implemented token-level explainability using SHAP PartitionExplainer (transformer) and
  DeepExplainer (LSTM), surfacing per-token attribution scores through a REST API endpoint
  — enabling model interpretability without a separate tooling stack

• Built a model drift monitoring pipeline using KL-divergence on a rolling 2,000-prediction
  window; automated alerting when distribution shift exceeds 0.1 threshold, enabling proactive
  retraining before accuracy degradation reaches production users

• Conducted systematic architecture comparison (Bi-LSTM, DistilBERT, RoBERTa) with full
  MLflow experiment tracking, evaluating accuracy, F1, ROC-AUC, and P50/P95/P99 inference
  latency — documented tradeoffs in a decision framework adopted as team-facing documentation
